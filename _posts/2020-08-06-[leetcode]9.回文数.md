```
判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
```

> 注意会有负数的情况，负数，个位数，10 都不是回文数。其他的整数再按照回文的规则判断。

```golang
func isPalindrome(x int) bool {
	if x < 0 {
		return false
	}
	if x < 10 {
		return  true
	}

	s := strconv.Itoa(x)
	for i := 0 ; i <= len(s)/2 ; i ++ {
		if s[i] != s[len(s)-1-i] {
			return false
		}
	}
	
	return true
}
```
