# FileBeat
---
**简单使用**

配置文件:
```
filebeat.inputs:    //输入
- type: stdin  
  enabled: true 
setup.template.settings: 
  index.number_of_shards: 3   //转发到es中的分片个数
output.console:   //输出
  pretty: true  
  enable: true
```

```
./filebeat -e -c itcast.yml
```

**修改输入类型为文件**
```
filebeat.inputs:
- type: log               //输入类型
  enabled: true
  paths: 
    -  /home/chenhu/logs/*.log   //文件路径
setup.template.settings:
  index.number_of_shards: 3
output.console:
  pretty: true
  enable: true

```

**加上tag和自定义字段**

```
filebeat.inputs:
- type: log
  enabled: true
  paths:
    -  /home/chenhu/logs/*.log
  tags: ["web","test"]   //tag 数组
  fields:  #添加自定义字段   
    from: itcast-im  
  fields_under_root: true #true为添加到根节点，false为添加到子节点中 
setup.template.settings: 
  index.number_of_shards: 3 
output.console:  
  pretty: true  
  enable: true

```
